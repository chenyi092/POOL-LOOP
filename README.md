# POOL LOOP
## プロジェクト概要 | Project Overview
POOL LOOP は、情報爆発の時代において「役に立たない情報」がどのように社会のノイズとなり、そしてどのように再生し得るかを問いかける、体験型のインタラクティブ・インスタレーションです。
> [!NOTE]
> **開発期間**：2023年10月 - 2023年12月（約3ヶ月）<br>
**チーム構成**：4名<br>
**自分の役割**：メインプログラマー（Unity）、システムアーキテクチャ設計、ハードウェア制御ロジック<br>
**主な技術***：Unity, C#, Firebase, p5.js, Raspberry Pi, TextMesh Pro<br>
**作品ジャンル**：インタラクティブ・インスタレーション
## 作品コンセプト | Concept
現代社会は、無価値な「情報ゴミ」が溢れかえる海のようなものです。<br>
本プロジェクトは、実世界のゴミがリサイクルされるのと同様に、ネット上の無用な情報も再び「価値」や「エネルギー」へと変換できるのではないか、という仮説に基づいています。 <br>
以下の3つのキーワードに対する問いを目的としています
  - 情報の過剰摂取 
  - デジタル・リサイクル
  - エネルギーの可視化
